## Project Overview

A shared library suite for Spring Boot 3 / Jakarta EE 10 / JDK 21+, designed to
accelerate development and reduce boilerplate code in business applications.

- **Java**: 21
- **Build tool**: Maven
- **Main modules**: `ecuacion-splib-core`, `ecuacion-splib-web`, `ecuacion-splib-web-jpa`,
  `ecuacion-splib-jpa`, `ecuacion-splib-batch`, `ecuacion-splib-rest`
